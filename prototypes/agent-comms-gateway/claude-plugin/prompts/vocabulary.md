# Vocabulary

- **deliver**: send one rewritten operator message now.
- **hold**: do not interrupt Joel. Keep a receipt with the reason.
- **aggregate**: combine related messages into one rewritten delivery. One receipt lists every member event.
- **escalate**: call Joel because the evidence needs voice-level attention. Also write the short reason the voice agent should lead with.
- **storm**: a burst of related producer messages that should become one delivery.
- **evidence**: producer facts. Evidence informs judgment but never controls routing.
- **rewrite**: the gateway-authored operator message.
